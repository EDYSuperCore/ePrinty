# prnport.vbs 脚本使用说明

## 脚本位置

请将 `prnport.vbs` 文件放在 `src-tauri/scripts/` 目录下。

## 当前实现

代码会使用以下方式调用脚本：

```bash
cscript //NoLogo //B scripts/prnport.vbs -a -r IP_192_168_20_65 -h 192.168.20.65
```

### 参数说明：
- `-a`: 添加端口
- `-r`: 端口名称（格式：IP_192_168_20_65）
- `-h`: 打印机 IP 地址（格式：192.168.20.65）

## 如果你的 prnport.vbs 使用不同的参数格式

如果你的 `prnport.vbs` 脚本使用不同的参数格式，请告诉我，我会相应调整代码。

常见的变化：
1. **参数顺序不同**：例如 `-r` 和 `-h` 的顺序
2. **参数名称不同**：例如使用 `/a` 而不是 `-a`
3. **参数格式不同**：例如不需要 `-r` 参数，IP 地址直接作为参数

## 脚本调用流程

1. **添加打印机端口**：使用 `prnport.vbs` 添加 TCP/IP 端口
2. **添加打印机**：使用 PowerShell `Add-Printer` 命令添加打印机
